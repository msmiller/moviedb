# README

Built using the following ...

- TailwindCSS
- Hotwire/Turbo
- ViewComponent
- ImportMap/CSS-Bundling _(no mo' WebPacker)_

## Install / Run

1. git clone https://github.com/msmiller/moviedb.git
2. cd moviedb
3. bundle
4. yarn
5. rails db:migrate
6. Set `TMDB_KEY` in `.env`
7. foreman start
